🚀 COMO COMEÇAR (SIGA ESTES PASSOS):

1️⃣ INSTALAR POSTGRESQL
   • Baixe: https://www.postgresql.org/download/windows/
   • Durante instalação, anote a senha do usuário 'postgres'

2️⃣ ABRIR POWERSHELL E EXECUTAR:

   cd "c:\Users\Sueli\Desktop\BRUNO"
   
   # Instalar dependências
   npm install
   
   # Criar banco de dados (se ainda não criou)
   # psql -U postgres
   # CREATE DATABASE barbearia_luxo;
   # \q
   
   # Aplicar migrations e popular dados
   npx prisma migrate dev --name init
   node prisma/seed.js

3️⃣ INICIAR O SISTEMA (ABRIR 2 TERMINAIS):

   Terminal 1:
   npm run dev
   
   Terminal 2:
   npm run worker

4️⃣ ACESSAR NO NAVEGADOR:
   http://localhost:3000
